### Assistant

Docker의 `ENTRYPOINT`를 말씀하시는 건가요?

**ENTRYPOINT**는 Dockerfile에서 컨테이너가 시작될 때 실행할 기본 명령을 지정하는 명령어입니다.

```dockerfile
ENTRYPOINT ["python", "runner.py"]
```

**CMD와의 차이:**

| | `ENTRYPOINT` | `CMD` |
|---|---|---|
| 역할 | 실행할 **프로그램** 고정 | 기본 **인자** 제공 |
| 덮어쓰기 | `--entrypoint` 플래그로만 가능 | `docker run` 시 인자로 덮어쓰기 가능 |

**함께 쓰는 패턴:**
```dockerfile
ENTRYPOINT ["python", "runner.py"]
CMD ["--default-arg"]
```
→ `docker run image`는 `python runner.py --default-arg` 실행  
→ `docker run image --other-arg`는 `python runner.py --other-arg` 실행
