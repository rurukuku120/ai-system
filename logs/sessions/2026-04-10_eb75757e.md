## 개념 정리

**에이전트** = 단일 역할, 단일 입출력
```
asset-parser: .uasset 파일 → 구조화된 데이터
vfx-evaluator: 이미지 → 평가 JSON
notion-writer: JSON → Notion 페이지
```

**하네스** = 에이전트들을 순서대로 엮은 파이프라인
```
vfx-evaluator (하네스)
  └─ 1. inbox/ 스캔
  └─ 2. Claude로 이미지 평가 → JSON
  └─ 3. notion-writer 호출 → Notion 등록
  └─ 4. slack-notifier 호출 → 결과 알림
```
