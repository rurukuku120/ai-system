## Claude Code 규칙

- 항상 한국어로 응답한다.
- PowerShell 환경에서는 PowerShell 문법을 사용한다.
- 기존 사용자 변경을 임의로 되돌리지 않는다.
- 외부 아티팩트 생성은 명시 요청이 있을 때만 수행한다.
