# Fit Coach

식단 사진, 이분할 헬스 루틴, 러닝 앱 스크린샷을 한곳에 모아 종합 코칭을 보여주는 정적 웹앱 프로토타입이다.

## 실행

`index.html`을 브라우저에서 열면 바로 사용할 수 있다.

AI 분석 키를 브라우저에 넣지 않고 테스트하려면 다음처럼 실행한다.

```powershell
$env:OPENAI_API_KEY="sk-..."
node server.js
```

그 다음 `http://localhost:4173`을 연다.

## 기능

- 식단 사진 업로드와 음식 메모 기반 영양 균형 추정
- 상체/하체 이분할 헬스 루틴 기록
- 러닝 앱 스크린샷 업로드와 거리, 시간, 심박 기반 페이스 분석
- 식단, 헬스, 러닝을 종합한 오늘의 수정 제안
- `server.js` 또는 OpenAI API Key가 있으면 사진과 스크린샷을 `Responses API`로 분석
- 브라우저 `localStorage` 기반 기록 저장

## 현재 범위

API 키가 없거나 분석 방식이 `로컬만`이면 사용자가 입력한 음식 메모와 러닝 수치를 바탕으로 코칭 결과를 생성한다.
브라우저에 직접 API 키를 넣는 방식은 로컬 프로토타입용이다. 실제 배포 버전은 `server.js`처럼 서버 프록시에서 키를 보호해야 한다.
